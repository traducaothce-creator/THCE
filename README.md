# THCE
Tradução Histórica e Científica das Escrituras (THCE)

 A Tradução Histórica e Científica das Escrituras (THCE) é um projeto independente de tradução bíblica cujo objetivo é oferecer uma versão que priorize fidelidade filológica, coerência terminológica e respeito ao contexto histórico dos textos originais. A proposta da THCE é reduzir interferências interpretativas posteriores ao texto e preservar, sempre que possível, as estruturas gramaticais e as nuances semânticas presentes nos idiomas originais.
 A tradução é realizada com base no texto massorético hebraico para o Antigo Testamento e no texto crítico grego para o Novo Testamento, com consulta a léxicos acadêmicos reconhecidos e análise sintática contextual. O projeto busca evitar harmonizações doutrinárias e inserções interpretativas não exigidas pelo próprio texto, mantendo distinção clara entre tradução e interpretação.
 A THCE preserva termos técnicos quando necessário, como Sheol, Geena e Hades, e mantém a distinção do Nome Divino (YHWH, representado como Yauyé) como parte de sua proposta de transparência histórica. Ambiguidades presentes no texto original são mantidas quando a gramática permitir mais de uma leitura legítima, evitando resoluções baseadas em pressupostos teológicos posteriores.
 O projeto está em desenvolvimento contínuo, com tradução progressiva dos 66 livros canônicos. A meta é concluir todo o corpus mantendo uniformidade metodológica e consistência lexical. Após a conclusão integral, está prevista uma revisão terminológica global e o desenvolvimento de uma edição acadêmica com aparato crítico ampliado.
